maos_phase_screen_grid¶

paarti.utils.maos_utils.maos_phase_screen_grid(r0s, l0s, on_sky, thres=0.05)[source]¶

Function to test multiple NCPA r0 and l0 parameter values, in comparison to an on-sky Strehl quantity (currently hard-coded below). The best (r0, l0) combinations are determined via the input threshold.
